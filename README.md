# Student_CRM

*学员管理系统，应用xadmin*

# 说明
纯学习练手项目，安装步骤如下：

**下载：**
```shell
$ git clone https://github.com/zhu-jie/Student_CRM.git
```

**安装**
```shell
$ cd Student_CRM/
$ pip install -r requirements.txt
```

**同步数据库**
```shell
$ python manage.py makemigrations
$ python manage.py migrate
```

**运行项目**

```shell
$ python manage.py runserver 0:8080
```
